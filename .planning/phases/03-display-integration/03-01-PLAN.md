    `export function generateInsights(data: DashboardData) => Insight[]`

    Where Insight is `{ type: 'positive'|'warning'|'danger'|'info', icon: string, title: string, body: string }`.

    **Implementation logic:**

    1. Determine `currentYear` = max year across all pnl metrics. `priorYear` = currentYear - 1.

    2. **Growth highlights** -- For each non-excluded company in `data.pnl`:
       - Sum Revenue for currentYear and priorYear
       - If priorYear revenue > 0 and growth > 50%: push `{ type: 'positive', icon: '\uD83D\uDCC8', title: '${name} revenue grew ${pct}% YoY', body: 'Revenue increased from ${fmt(prior)} to ${fmt(current)}.' }`
       - If priorYear revenue === 0 and currentYear revenue > 0: push `{ type: 'info', icon: '\uD83C\uDF31', title: '${name} started generating revenue', body: '...' }`

    3. **Profitability milestones** -- For each non-excluded company:
       - Sum EBITDA for currentYear and priorYear
       - If priorYear EBITDA < 0 and currentYear EBITDA >= 0: push `{ type: 'positive', icon: '\uD83C\uDFC6', title: '${name} reached EBITDA breakeven', body: 'EBITDA improved from ${fmt(prior)} to ${fmt(current)}.' }`

    4. **Risk flags** -- For each non-excluded company:
       - If currentYear EBITDA < 0 AND revenue growth > 20%: push `{ type: 'warning', icon: '\u26A0\uFE0F', title: '${name} losses persist despite revenue growth', body: 'Revenue grew ${pct}% but EBITDA remains ${fmt(ebitda)}. Cost structure may need attention.' }`
       - If currentYear EBITDA < -1000000 (losses > $1M): push `{ type: 'danger', icon: '\uD83D\uDD34', title: '${name} has significant losses', body: '...' }`

    5. **Cash runway** -- From `data.cashflow`:
       - Sum 'Net Cash Flow' across all companies for all months to get total net cash movement
       - Get last month's cumulative balance (sum all 'Net Cash Flow' values = net movement, but need ending balance). Use sum of last month's 'Cash Inflow' - abs('Cash Outflow') values as monthly net, then calculate average monthly burn = totalNetCashMovement / monthCount.
       - Actually simpler: sum all 'Net Cash Flow' values across all companies across all months = totalNetMovement. monthCount = number of distinct months. avgMonthlyBurn = totalNetMovement / monthCount. endingBalance = determine from running total.
       - If avgMonthlyBurn < 0: runwayMonths = endingBalance / Math.abs(avgMonthlyBurn)
       - If runwayMonths < 6: type='danger', if < 12: type='warning', else: type='info'
       - Push `{ type, icon: '\uD83D\uDCB0', title: 'Cash runway: ~${months.toFixed(0)} months', body: 'Ending cash balance of ${fmt(balance)} with average monthly burn of ${fmt(burn)}.' }`
       - If avgMonthlyBurn >= 0 (cash positive): push info insight about positive cash flow

    6. Return the insights array (no fixed ordering needed -- categories naturally group).

    Helper: create a private `sumMetricForYear(company, metricName, year)` function to avoid repetition.
